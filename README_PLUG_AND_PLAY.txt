WILDLANDS WEB DEMO (PLUG & PLAY)

EASIEST (NO SERVER):
  - Double-click: PLAY_WILDLANDS.html
  - Click: "PLAY — Start Game"
  - This build bundles all Bible data so it runs from file:// without CORS issues.

OPTIONAL (LOCAL SERVER, BEST COMPATIBILITY):
  Windows: double-click start_server_windows.bat
  macOS/Linux: run ./start_server_mac_linux.sh then open http://localhost:8000

Controls:
  WASD / Arrows = move
  Mouse = aim
  Left click = shoot
  Space = dash
  Q = active ability
  R = restart
  1–3 = pick evolution node (level-up)
